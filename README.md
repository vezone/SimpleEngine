# **Simple Engine**

## **About**
Simple game engine written in C for 2D games creation.

## **Platforms supported**
* GNU/Linux
* Windows

## **Dependencies**
### GNU/Linux
* premake5


## **Getting started**

#### GNU/Linux
* git clone https://github.com/vezone/SimpleEngine
* cd SimpleEngine/Scripts/Linux
* ./generate_project.sh
* ./build_script_debug.sh

#### Windows (WIP)
* git clone https://github.com/vezone/SimpleEngine
* cd SimpleEngine/Scripts/Linux
* ./generate_project.bat
* ./build_script_debug.bat
